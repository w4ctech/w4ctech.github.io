    Flutter开发环境搭建

    # 安装Flutter

    1. git clone https://github.com/flutter/flutter.git
    2. 在系统变量path新增 E:\flutter\bin
    3. 新增PUB_HOSTED_URL系统变量https://pub.flutter-io.cn
    4. 新增FLUTTER_STORAGE_BASE_URL系统变量https://storage.flutter-io.cn
    5. cd flutter
    6. flutter doctor

    # Flutter开发必备其他环境

    1. Java Jdk https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
    2. Android Studio https://developer.android.com/studio/
    3. Git https://git-scm.com/download/win

    # 添加Flutter插件

    1. 打开Android Stuido,然后找到Plugin的配置，搜索Flutter和Dart插件并安装。
